# Life and Miracles of Jesus

## Overview
This project is an assignment for COMP30680 Web Application Development. The task was to create a website using HTML and CSS, ensuring valid code and a structured design. The website explores the life and miracles of Jesus Christ, providing information and multimedia elements to engage users.

## Features
The website meets the assignment requirements by including:
- **Five Pages**:
  - `index.html` (Home Page)
  - `gallery.html` (Gallery Page)
  - `M1Water_Wine.html` (Water into Wine)
  - `M2Nobleman_son.html` (Healing of the Noblemanâ€™s Son)
  - `M3Healing_Blind.html` (Healing the Blind)
- **Navigation Bar**: Links to all pages for easy navigation.
- **Links**:
  - Internal links to other pages within the website.
  - External links to Bible resources such as Bible Gateway, Bible.org, and Theopedia.
- **Table**: A structured table listing 37 miracles of Jesus with references.
- **Lists**: Ordered and unordered lists used for structuring information.
- **Embedded Video**: A YouTube video providing a summary of Jesus' life.
- **CSS3 Elements**:
  - `box-shadow`
  - `border-radius`
  - `opacity`
  - `flexbox`
  - `grid`
- **HTML5 Elements**:
  - `header`
  - `footer`
  - `nav`
  - `section`
  - `article`
- **CSS Positional Properties**:
  - `position: relative`
  - `position: absolute`
  - `float`
  - `display: flex`

## Additional Enhancements
To go beyond the basic requirements, the project includes:
- **Responsive Design**: Uses `@media` queries to improve mobile usability.
- **Hover Effects**: Enhances user experience with subtle animations.
- **Custom Styling**: Advanced CSS techniques for better readability and aesthetics.

## Validation
- HTML validated using [W3C HTML Validator](https://validator.w3.org/).
- CSS validated using [W3C CSS Validator](https://jigsaw.w3.org/css-validator/).

## Usage
1. Clone the repository:

